СПИСОК КОМАНД ДЛЯ РАБОТЫ С БОТОМ:
/start - запуск
/menu - начало работы
/share - поделиться 
/help - если непонятно как работать с ботом
/get - получение статистики
/news - рассылка новостей
/adminMenu - вызов меню программирования
/adminMenuCancel - вызов команды отмены меню программирования
/card - получить карту партнера
/user - команда для отправки пользователю сообщения. /user 12344556=ID сообщение
/render - команда запуска таймеров рендера меню по времени, в базе данных
/render_cls - команда отмены таймеров рендера
/cmd - вызов справки по командам
